                  What's it for?

                  The Character Creation Utility is for creating characters for Monte Cook Games's Numenéra pen-and-paper role-playing game.

                  How does it work?

                  It is an all-Javascript/HTML/CSS webapp written with the Dojo Toolkit.

                  Do you collect any data?

                  No, other than usual web server traffic logging. If you link to characters you've shared, that request and its query string does get logged in the web server's HTTP logs.

                  So where do my saved characters go?

                  In your browser's local data store. That means they'll only be available on the browser you used to create them, unless you've set up a service that syncs your browser data between devices.

                  Will I lose the characters if I empty my history, clear my cache, and remove my cookies?

                  The details depend on your browser, but yeah, you most likely will.

                  Any way I can "back up" my characters?

                  Bookmark the "Share (character name)" link. That won't help if I've updated the app to break data compatibility again. The primary purpose of this app (at this point) is to easily fill out and print a character sheet; that's your real backup.

                  Anything else about my saved characters?

                  Don't trust this app. I'm updating it frequently and an update may break all your characters without warning. Once this becomes stabler, I'll post a notice on Ninth World Hub.

                  Any gotchas with the printable character sheet?

                  It seems to work OK on Firefox and Google Chrome, but Firefox prints ugly headers and footers on it. For best results, use Chrome.

                  I have an idea. Where can I reach you?

                  You'll find me on Ninth World Hub, as Petteri Sulonen. I won't leave my e-mail here because spam and stuff.

                  Is this an official Numenéra product?

                  No way! Numenera is © Monte Cook Games LLC. The Character Creation Utility is an unofficial fan project by me, Petteri Sulonen. I may take it off-line at any time, for any reason, in particular if MCG asks me to, so don't rely on it too much.
