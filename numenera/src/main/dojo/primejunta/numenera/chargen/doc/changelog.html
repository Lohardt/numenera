Version History

2.2.0

  • Character roster in play view. Add/remove characters from it and switch between them with tab bar buttons.

2.1.1

  • Notes and description are now editable in play view, and synced back to the main character sheet on close.

2.1.0

  • New feature: download and restore character data from local file. Open the character manager from the -button.
  • XP set to zero when clearing a character and starting a new one, not only when first advancing the new character.

2.0.1

  • In in-play view, XP is displayed as 0 for non-finalized characters (was "false")
  • Re-enabling disabled inventory items works again
  • Loading a previously saved character from a link works again
  • Character data now saved to undo buffer more frequently

2.0.0

  • In-play utility for easy display of character data and tracking stat pools
  • Major update to the underlying architecture
  • When exploring descriptors, types, and foci, only clears the minimum necessary
  • Nicer transition FX
  • A few more portraits
  • Converts portraits to JPEG format for smaller file sizes
