1.0.0-b20

1.0.0-b19

  • Puts back those nice fonts since I figured out how to cache them
  • Makes attack filtering case-insensitive, to make life easier for self-added weapons
  • Properly filters out -- choose -- items from inventory, for character sheet

1.0.0-b18

  • Replaced nice web fonts with standard equivalents for now, because they do not cache reliably and break the offline features

1.0.0-b17

  • Includes items from Extra Equipment when collecting attacks
  • Marks Death Dealer as Action
  • Improves printed sheet, especially for high-tier characters with big lists
  • CSS cleanup and documentation improvements

1.0.0-b16

  • Fixes bug with advancement control not appearing when it should when unloading and loading finalized characters
  • Moves skills gained as Tricks of the Trade into their proper place (and looks for resulting specializations)
  • Handles repeating Enablers (e.g. Reduce Armor Cost) more intelligently
  • Puts default in Fuses Flesh and Steel perk Weaponization
  • Fixes bug that puts garbage in an input field if no default is specified
  • Fixes bug in the way attack bonuses were calculated; light weapons benefited one notch too much
  • Defines attack and defense bonuses from feats as data rather than embedded in the logic

1.0.0-b15

  • Improves validation of stored character data to detect e.g. corrupt links
  • Removes dedicated list of connections to other PC's, and puts that information in description instead where it can be edited to taste
  • Updates link and enables save button more frequently
  • Many cosmetic changes

1.0.0-b14
